Программа для раздления с сортировки объемного файла.

Функция генерации файла со случайными значениями вынесена в отдельный файл filegen.cpp
Для сортировки созданного файла используется filesort.cpp вместе с qsort.h и qsort.cpp

После открывания сгенерированного файла, данные считываются в массив, разделяются на две части и записываются во временные файлы. Затем эти данные упорядочитваются с помощью быстрой сортировки и сливаются в один итоговый файл с помощью сортировки слиянием.
